// 1. How to apply css ?
----------------------------------------------


how to apply style to HTML elements ?
    
    3 ways

    way-1 : inline style
    -----------------------

    syntax  :

    style ="style-property1:value;style-property2:value"

    way-2 : internal style
    -----------------------

    syntax :

    <head>
        <style>
        </style>
    </head>


    way-3 : external style  ( recommended )
    ----------------------------------------

    syntax:
    --------
     
    <head>
        <link href="style.css" rel="stylesheet"/>
    </head>
